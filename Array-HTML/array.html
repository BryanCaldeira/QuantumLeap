What is an Array?

— An Array is a sequential collection of elements of same data type.

— Array stores elements in continuous memory location.

— Array elements can be accessed by using Index

— The range of Index of an array of size 'N' is in the range '0' to 'N-1'

— For example, if we have a array of 5 elements, then the array index will be in the range of '0' to '4'

— Any element in the array can be accessed by using array[index]

— From the above example array we can access the element 10 by using array[2]

Array Declaration

— In C/C++

type  arr[5];

— In Java

type  arr[ ] = new  type[size];


— Here 'type' can be any datatype supported by the language
ex:  char,  int,  float,  bool, ...etc

— In Java the declaration looks a bit different because it includes declaration and memory allocation in a single step

— The elements in the array allocated by new will automatically be initialized to zero (for number types), false (for boolean), or null for reference types

Array Initialization

— Array can be initialized either at the time of declaration or after that

— In C/C++

type  arr[size] = {elem 1, elem 2, ..., elem n};

— In Java

type  arr[ ] = new  type[ ]{elem 1, elem 2, ..., elem n};


— Here 'type' can be any datatype supported by the language
ex:  char,  int,  float,  bool, ...etc
